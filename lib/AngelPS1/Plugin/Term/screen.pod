=head1 NAME

AngelPS1::Plugin::Term::screen - Utilities for C<GNU screen>

=head1 SYNOPSIS

Static C<screen> title:

    use AngelPS1::Plugin::Term::screen;

    my @prompt = (
	...
	ScreenTitle($title),
	...
    )

Dynamic C<screen> title:

    use AngelPS1::Plugin::Term::screen;

    my @prompt = (
	...
	ScreenTitle(sub {
	    return scalar localtime
	}),
	...
    )

=head1 DESCRIPTION

Set the title of the C<GNU screen> session every time the
prompt is displayed.

The C<ScreenTitle> takes a list as arguments and wraps it to direct the
expanded content as the name of the screen session.
